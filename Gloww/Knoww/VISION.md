# Knoww — Future Vision

This doc captures the dream features for future phases. Not everything needs to happen — this is the wishlist.

---

## Phase 2: The Interactive Layer

### Infi Character
- A 2D illustrated character (Lottie-animated) representing Indira
- Name: "Infi"
- Appears on the landing page as a guide
- Idle animations while reading (sitting in corner, swaying legs, like VS Code pets)
- Different expressions/poses per section

### Visitor Character
- Visitors can choose a simple avatar (preset skins/colors, not a full builder)
- Both characters appear together on pages
- They "interact" via idle animations (talking, waving, etc.)

### Interactive Choice Prompts
- Clickable doodle-style choices: "SDE side or Product side?"
- Each choice leads to different content paths
- Flying-papers style interaction (inspired by flying-papers-923 on Godly)

### Guided Tour Mode
- "Infi takes you around" — a step-by-step walkthrough of the site
- Creative name for the tour TBD
- Optional: visitors can skip the tour and explore freely

---

## Phase 3: The Full Capsule Dream

### Full 3D Experience
- Three.js scenes on key pages (not just particles)
- Atmos/Wayfinder-style creative navigation
- 3D elements respond to cursor movement and scroll

### Personalization Flow
- Inspired by My Stella personalization
- Visitors customize their experience (color theme? interests?)
- Site adapts based on choices

### Advanced Animations
- Page transitions with morphing elements
- Parallax depth on scroll
- Sound design (subtle, optional — toggle on/off)

## Notes
- All features must stay within free-tier constraints
- Static site / GitHub Pages compatible
- Mobile-first — every feature must work on touch
- Keep the personality: playful, casual, energetic — never corporate
